1. **Install dependencies:**
   ```sh
   npm install
   ```
2. **Konfigurasi database di `.env`**
3. **Generate Prisma Client:**
   ```sh
   npx prisma generate
   ```
4. **Migrasi database:**
   ```sh
   npx prisma migrate dev --name init
   ```
5. **Jalankan server:**
   ```sh
   npm start
   ```
